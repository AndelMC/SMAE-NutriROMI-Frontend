# SMAE-NutriROMI Frontend

Este es el frontend de SMAE-NutriROMI, una interfaz moderna diseñada para consultar el Sistema Mexicano de Alimentos Equivalentes (SMAE). Desarrollado con Next.js y Tailwind CSS.

## 🚀 Producción

La aplicación está desplegada en Vercel:
- **Enlace:** [https://smae-nutri-romi.vercel.app/](https://smae-nutri-romi.vercel.app/)

## 🛠️ Desarrollo Local

Para correr el proyecto localmente:

1. Instala las dependencias:
   ```bash
   npm install
   ```

2. Ejecuta el servidor de desarrollo:
   ```bash
   npm run dev
   ```

3. Abre [http://localhost:3000](http://localhost:3000) en tu navegador para ver el resultado.

*Nota: Asegúrate de tener el [backend](https://github.com/AndelMC/SMAE-NutriROMI-Backend) corriendo en el puerto 3001 si deseas probar la API localmente, o utiliza las variables de entorno correspondientes.*
